### 🔄 방안 변경: curl을 통한 Java 21 설치 성공!
<!-- ✅ COMPLETED: Java 21 설치 성공 -->

#### Java 21 설치 과정:
```bash
# 1. curl로 OpenJDK 21 다운로드
curl -L -o openjdk-21.zip "https://download.java.net/java/GA/jdk21/fd2272bbf8e04c3dbaee13770090416c/35/GPL/openjdk-21_windows-x64_bin.zip"
결과: 191M 다운로드 성공 (30.3M/s 속도)

# 2. 사용자 폴더에 압축 해제
powershell Expand-Archive -Path 'openjdk-21.zip' -DestinationPath '%USERPROFILE%\Java21'
결과: C:\Users\Im\Java21\jdk-21에 설치 완료

# 3. 환경변수 설정
set JAVA_HOME=%USERPROFILE%\Java21\jdk-21
set PATH=%JAVA_HOME%\bin;%PATH%

# 4. 버전 확인
java -version: OpenJDK 21+35-2513 ✅ 성공!
```

### 4단계: Java 21로 Spring Boot 재실행 시도

#### 환경변수 설정 (PowerShell):
```powershell
$env:JAVA_HOME = "$env:USERPROFILE\Java21\jdk-21"
$env:PATH = "$env:USERPROFILE\Maven\apache-maven-3.9.5\bin;$env:JAVA_HOME\bin;$env:PATH"

# 확인
java -version: OpenJDK 21+35-2513 ✅
mvn -version: Apache Maven 3.9.5 ✅
```

### ❌ 실행 실패 - 네트워크 및 플러그인 문제 발견

#### 실행 결과:
```
[INFO] BUILD FAILURE
[ERROR] No plugin found for prefix 'spring-boot' in the current project
원인: 네트워크 연결 문제로 Maven Central 저장소 접근 실패
```

#### 발견된 문제들:
1. **네트워크 차단**: `maven-default-http-blocker` 설정으로 Maven Central 차단됨
2. **프록시 문제**: `Connect to 70.10.15.10:8080 failed: Connect timed out`
3. **Spring Boot 플러그인 누락**: pom.xml의 spring-boot-maven-plugin 인식 실패
